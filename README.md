## 📖 SOBRE O REPOSITÓRIO

Este repositório foi criado com o objetivo de documentar minha jornada de aprendizado em **Python**, reunindo exercícios, exemplos práticos e pequenos projetos desenvolvidos ao longo de um curso introdutório.

A ideia é servir tanto como material de estudo quanto como um histórico da minha evolução na linguagem.

## 🎯 OBJETIVOS

- 📚 Aprender os fundamentos da linguagem Python  
- 🧠 Desenvolver lógica de programação  
- 🛠️ Praticar com exercícios e pequenos projetos  
- 📈 Evoluir gradualmente até conceitos mais avançados  

## 🚀 CONTEÚDOS ABORDADOS

O repositório está organizado com base nos principais tópicos estudados:

### 🧩 Fundamentos
- Variáveis e tipos de dados  
- Entrada e saída de dados (`input` / `print`)  
- Operadores (aritméticos, lógicos e relacionais)  

### 🔀 Estruturas de Controle
- Condicionais (`if`, `elif`, `else`)  
- Estruturas de repetição (`for`, `while`)  

### 📦 Estruturas de Dados
- Listas  
- Tuplas  
- Dicionários  
- Strings  

### 🧪 Funções
- Criação de funções  
- Parâmetros e retorno  
- Reutilização de código
